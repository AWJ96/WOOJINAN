Steady-State Analysis Tool for Modular Multilevel Converters in VSC-HVDC Systems (On-going)

Institution: Imperial College London
Supervisor: Dr. Adria Junyent-Ferre
Tools & Techniques: MATLAB, GUI Development, Converter Modeling, Signal Processing, Power Electronics

This project developed a MATLAB-based tool for the steady-state analysis of Modular Multilevel Converters (MMCs) used in Voltage Source Converter–High Voltage Direct Current (VSC-HVDC) systems. The tool allows users to configure converter parameters and visualize key internal dynamics such as arm currents, capacitor voltage ripple, and submodule energy. It also calculates and displays feasible active/reactive power boundaries, supporting converter control design and operating point validation.

Key Contributions

  • Designed and implemented a GUI-based MATLAB tool for steady-state MMC simulation
  • Modeled internal converter behavior including submodule capacitance, circulating currents, and power exchange
  • Enabled parametric sensitivity analysis (e.g., effects of AC-side filter impedance and SM capacitance)
  • Visualized converter P–Q operating regions for assessing steady-state feasibility
  • Established a foundation for future integration of transient and fault-response analysis
